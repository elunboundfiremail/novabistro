PASOS PARA EJECUTAR EL PROYECTO:

1. Crear la base de datos en PostgreSQL:
   psql -U postgres
   CREATE DATABASE novabistro;
   \q

2. Ejecutar el script de creacion de tablas:
   cd F:\datafiles\hckrmn\Carrera\Carrera\tecnologias web1\PROYECTO\novabistro
   psql -U postgres -d novabistro -f base_datos\schema.sql

3. Instalar dependencias con UV:
   uv sync

4. Ejecutar la aplicacion:
   uv run uvicorn main:app --reload

5. Acceder a la API:
   - API: http://127.0.0.1:8000
   - Documentacion Swagger: http://127.0.0.1:8000/docs
   - ReDoc: http://127.0.0.1:8000/redoc

ENDPOINTS DISPONIBLES (31 TOTAL):

ROLES (/roles):
- GET /roles                - Listar todos los roles
- POST /roles               - Crear nuevo rol
- PUT /roles/{id}           - Actualizar rol
- DELETE /roles/{id}        - Eliminar rol (soft delete)

PERSONAL (/personal):
- GET /personal             - Listar todo el personal
- POST /personal            - Crear nuevo personal
- PUT /personal/{id}        - Actualizar personal
- DELETE /personal/{id}     - Eliminar personal (soft delete)

MESAS (/mesas):
- GET /mesas                - Listar todas las mesas
- POST /mesas               - Crear nueva mesa
- PUT /mesas/{id}           - Actualizar mesa
- DELETE /mesas/{id}        - Eliminar mesa (soft delete)

CATEGORIAS (/categorias):
- GET /categorias           - Listar todas las categorias
- POST /categorias          - Crear nueva categoria
- PUT /categorias/{id}      - Actualizar categoria
- DELETE /categorias/{id}   - Eliminar categoria (soft delete)

PRODUCTOS (/productos):
- GET /productos            - Listar todos los productos
- POST /productos           - Crear nuevo producto
- PUT /productos/{id}       - Actualizar producto
- DELETE /productos/{id}    - Eliminar producto (soft delete)

PEDIDOS (/pedidos):
- GET /pedidos              - Listar todos los pedidos
- GET /pedidos/estado/{estado} - Filtrar pedidos por estado
- POST /pedidos             - Crear nuevo pedido
- PUT /pedidos/{id}         - Actualizar pedido
- DELETE /pedidos/{id}      - Eliminar pedido (soft delete)

- roles (id_rol, nombre, descripcion, activo)

- categorias (id_categoria, nombre, descripcion, activo)
- productos (id_producto, nombre, descripcion, precio_bs, id_categoria, disponible, activo)
- pedidos (id_pedido, numero_pedido, id_mesa, id_personal, fecha_pedido, estado, total_bs, observaciones, activo)
- detalle_pedidos (id_detalle, id_pedido, id_producto, cantidad, precio_unitario_bs, subtotal_bs, observaciones)
